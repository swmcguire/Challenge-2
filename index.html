Class Project-1

Class Project-1 turned out to be an amazing learning experience. I got paired up with some great guys. We had some ups and downs but managed to put together a solid project. We designed a website that connects to a food and drink API that will populated food/drink choices based on ingredients you have lying around. Once you have a decision you can add that to a shopping list so you are ready to go. This project used APIs, HTML, CSS, and JavaScript to demonstrate what we've learned thus far.
